Relevant Projects and Publications

  • Universal Clocking off Rotating Neutron Stars:
    An initial mathematical analysis of pulsar spin down dynamics for use
    in time-keeping technology
    Description: Developed novel mathematics for characterizing a pulsar's spin-down. This paper outlines our approach to using known mathematical relationships and then generalizing them so as to better model pulsar spin-down for pulsars with assumed variable radiation-rotation angles so as to fit data pulled from any pulsar. This is then used to attempt to make predictions both about the pulsar's braking index and then to make timing predictions.

  • Human Freedom Index
    Description: Worked with two large datasets from different sources so as to graphically analyze various measures of human freedoms vs economic freedoms for over 150 countries. This project used RStudio to ingest, join, and transform these datasets with the output being various graphical representations of said data so as to find patterns within the large-scale structure of the information.
